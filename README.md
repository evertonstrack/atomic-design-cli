# Atomic Design CLI
CLI to create Angular components and modules with Atomic Design methodology.


## Pre-requirements

- Install [Yarn](https://yarnpkg.com/en/)
- Install Gluegun `yarn global add gluegun`


## How to run

In project folder: 
```
yarn
yarn link
```
